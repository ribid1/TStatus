# TStatus
Displays and announces the turbine status, received from the xicoy telemetry system 

You can use the app in combination with the dbdis app: https://github.com/ribid1/dbdis
    
  
### Installation:
* Copy the TStatus.lua or the TStatus.lc in the folder: \Apps
* Unpack the TStstus_Audio.zip and copy the files in the folder: \Audio
